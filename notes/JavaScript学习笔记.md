# JavaScript基础知识

## 变量声明

```javascript
let name = 'Chime Stone';
const age = 25;
var city = 'Beijing';
```

## 函数

### 箭头函数
```javascript
const add = (a, b) => a + b;
```

### 普通函数
```javascript
function multiply(a, b) {
    return a * b;
}
```

## 数组方法

- `map()` - 映射
- `filter()` - 过滤  
- `reduce()` - 归约
- `forEach()` - 遍历
