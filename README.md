# Chime Stone 个人网站

> 这是 Chime Stone 的个人网站，记录学习和生活的点点滴滴。

## 功能特性

- 📝 个人笔记管理
- 🎨 响应式设计
- 🌟 动画效果
- 📱 移动端适配
- 💾 本地存储

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage

## 项目结构

```
web/
├── index.html          # 主页
├── notes.html          # 笔记页面
├── assets/
│   ├── css/
│   │   └── main.css    # 主样式文件
│   ├── js/
│   │   └── main.js     # 主脚本文件
│   ├── img/
│   │   └── avatar.jpg  # 头像图片
│   └── json/
│       └── images.json # Bing壁纸配置
├── package.json        # 项目配置
├── .gitignore         # Git忽略文件
└── README.md          # 项目说明
```

## 本地运行

1. 克隆项目到本地
2. 在项目目录下运行：
   ```bash
   python -m http.server 8000
   ```
3. 打开浏览器访问 `http://localhost:8000`

## 特色功能

### 笔记管理
- 支持添加、删除笔记
